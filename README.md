# Travel

Travel é um site (fictício) de hotelaria carioca, que dispõe de vários serviços como Academia, Spa, Espaço Pet, Restaurante, entre outros.

## Tecnologias utilizadas:

- HTML
- CSS
- Javascript
- ReactJs
